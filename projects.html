Midi Computer Controller

Languages:
  • Bash
  • Python

MIDI devices are hardware devices that output data streams that can be interpreted by softwares as musical instruments. I developed this program to make use of a MIDI device I owned in order to control some Linux system functionalities that I found cumbersome.
In order to maximize performance I coded this primarily in Bash, and I took the opportunity to code the remainder using Python which was a new language to me at the time. As asynchronous paradigms in Bash make use of multiple threads I decided to instead use multiple files instead to improve readability to future prospective users.
The end result was a program that could run in the background and interpret dial turns and key presses as Bash commands.
